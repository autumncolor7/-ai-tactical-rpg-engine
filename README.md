# 鋼域遠征

一個用 Phaser 製作的簡易戰棋遊戲原型。

## 內容

- 主頁機庫介面
- 改造與零件庫
- 底部「進入下一關」按鈕
- 格子戰鬥場景
- 單位選取、移動、攻擊與敵方回合

## 執行方式

直接用瀏覽器開啟 `index.html` 即可。

遊戲透過 CDN 載入 Phaser：

```html
https://cdn.jsdelivr.net/npm/phaser@3.90.0/dist/phaser.min.js
```

## GitHub Pages

這是純靜態頁面，可以直接部署到 GitHub Pages。將 repository 的 Pages 來源設定為 `main` 分支根目錄後，`index.html` 會成為首頁。
